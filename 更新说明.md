## 🔧 使用说明

### 图片浏览
1. 进入"界别基本情况"页面
2. 查看"界别活动照片"部分
3. 点击图片可放大查看
4. 使用筛选按钮按界别分类浏览

### PDF查看
1. 进入"街道委员小组"或"星级评定"页面
2. 查看PDF文档部分
3. 使用缩放、翻页控制
4. 点击下载按钮保存文件

### 文件管理
1. 进入"文件浏览器"页面
2. 选择文件分类
3. 使用搜索和过滤功能
4. 点击文件进行预览
5. 使用"检查"按钮验证文件可读性

### 状态监控
1. 在首页查看"文件状态监控"
2. 查看文件可读性统计
3. 查看不可读文件列表
4. 使用"立即检查"按钮手动检查
