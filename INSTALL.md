# 安装和启动说明

## 系统要求
- Node.js 16.0 或更高版本
- npm 8.0 或更高版本

## 快速启动

### 方法1：使用启动脚本（推荐）
1. 双击 `启动项目.bat` 文件
2. 选择选项启动项目

### 方法2：手动安装和启动

#### 步骤1：安装依赖
```bash
# 进入项目目录
cd "D:\zhuomian\ddd\政协委员通-web"

# 安装依赖包
npm install
```

#### 步骤2：启动开发服务器
```bash
npm start
```

#### 步骤3：访问应用
打开浏览器，访问：http://localhost:3000

#### 步骤4：构建生产版本（可选）
```bash
npm run build
```

## 故障排除

### 常见问题

#### 1. 端口被占用
如果3000端口被占用，系统会自动使用其他端口。查看终端输出获取实际端口号。

#### 2. 依赖安装失败
```bash
# 清除npm缓存
npm cache clean --force

# 重新安装
npm install
```

#### 3. 项目无法启动
确保Node.js版本符合要求：
```bash
node --version
```
